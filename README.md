# 📝 To-Do List Project

Um gerenciador de tarefas simples e intuitivo, focado em uma experiência de usuário limpa e funcional. Este projeto foi desenvolvido para consolidar conhecimentos em manipulação do DOM e eventos com JavaScript puro.

## 🚀 Funcionalidades

* **Adição Dinâmica:** Criação de novas tarefas através do formulário.
* **Marcação de Conclusão:** Funcionalidade de "toggle" para riscar tarefas finalizadas.
* **Remoção de Tarefas:** Botão de exclusão para remover itens da lista instantaneamente.
* **Validação de Campo:** Uso do atributo `required` para evitar tarefas vazias.
* **Interface Limpa:** Uso de `autocomplete="off"` para manter o foco na aplicação.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica.
* **CSS3:** Estilização moderna com Flexbox, sombras e bordas arredondadas.
* **JavaScript:** Lógica de manipulação de elementos HTML e escuta de eventos.
